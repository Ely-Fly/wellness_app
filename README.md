# Soluna Wellness 🌙

[![Vercel](https://img.shields.io/badge/Live_Demo-Vercel-000000?style=for-the-badge&logo=vercel&logoColor=white)](https://wellness-app-xi-sandy.vercel.app/)


**Soluna Wellness** is a minimalist, AI-powered companion designed to help you understand the patterns in your mood and habits. Built with a calming "Soft-UI" aesthetic, it bridges the gap between daily logging and actionable mental health insights.

---

## ✨ Key Features

* **Daily Logging:** A seamless interface to track your mood, habits, and daily influences (Work, Sleep, etc.).
* **AI Insights:** Leverages the **Google Gemini API** to analyze your patterns and provide personalized wellness feedback.
* **Weekly Overview:** An interactive calendar dashboard to track your consistency and "streaks."
* **Responsive Design:** Optimized for a focused experience across mobile and desktop.

## 🛠️ Tech Stack

* **Frontend:** React with Vite
* **Language:** TypeScript
* **AI Integration:** Google AI Studio (Gemini API)
* **Deployment:** Vercel

## 🚀 Getting Started

<details>
<summary><b>Click to expand: Installation & Local Development</b></summary>

### 1. Clone the Repository
```bash
git clone [https://github.com/Ely-Fly/wellness-app.git](https://github.com/Ely-Fly/wellness-app.git)
cd wellness-app
```

### 2. Install Dependencies
```bash
npm install
```

### 3. Set up Environment Variables
Create a `.env.local` file in the root directory and add your Gemini API key:
```env
GEMINI_API_KEY=your-gemini-api-key-here
```

### 4. Run the App
```bash
npm run dev
```

## 🎨 Design Philosophy

Soluna Wellness embraces a **"Soft-UI"** aesthetic to create a calming user experience:
* **Soft Shadows:** Subtle shadows and rounded corners give elements a floating, tangible feel.
* **Natural Palette:** A muted, earthy color scheme inspired by nature to promote tranquility.
* **Minimalist Interface:** Clean layouts and thoughtful spacing reduce cognitive load, allowing users to focus on their reflections.
